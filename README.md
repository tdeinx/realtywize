# Realtywize

AI Command Center for Real Estate — built by HGIA Solutions.

**Live:** https://realtywize.netlify.app

## Structure

- `index.html` — marketing landing page
- `app/index.html` — the Realtywize command center app (served at `/app`)
- `netlify.toml` — hosting config and routes

## Deploy

Connected to Netlify. Every push to `main` auto-deploys.

Static HTML / CSS / JS. The app's AI features call the Anthropic Claude API.
